# Virtual DOM, Reconcilliation , Fibre

  createRoot creates a DOM like structure behind,compare its DOM with main DOM and updates only those values which updates in main.
  React Fibre is reimplementation of react core algo.features inclued the ability to pause, abort, reuse work as new features come.
  Reconcilliation (virtual DOM)is the high level  description of react algo.The react algo uses tree to diff one algo from another to determine which code is to change is called reconcilliation.

**To improve performance of lists in the Fibre , Keys are used.
